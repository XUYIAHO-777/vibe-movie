## Suggested GitHub Issues

Create these public issues so the roadmap is visible:

1. Add narrated demo video to README
2. Add fallback recommendations when AI or TMDB fails
3. Add API response validation tests
4. Split poster/result view into dedicated components
5. Add user login and personal collections
6. Improve accessibility for keyboard and screen-reader users

Suggested labels:

```txt

```

## Where To Share

- Reddit: `r/nextjs`, `r/webdev`, `r/SideProject`
- X / Twitter: post the demo GIF and live link
- LinkedIn: frame it as an AI-assisted portfolio project
- Product Hunt: only after adding a short demo video
- 掘金 / 小红书 / B站: focus on "AI + Next.js + mood-based movie recommendation"

## Star-Friendly README Rules

- Show the demo GIF above the fold.
- Keep setup steps short and copy-pasteable.
- Explain what makes the project different.
- Include screenshots, architecture, and roadmap.
- Keep commits clear and active.
